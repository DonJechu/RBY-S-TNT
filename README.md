
# RBYs TNT (Datapack)

[cite_start]A comprehensive Minecraft Datapack that expands the explosive arsenal of the game with **almost 10 new types of TNT**, each featuring unique survival crafting recipes and specialized destructive behaviors[cite: 1].

## 🚀 Key Features

* [cite_start]**Diverse Explosive Arsenal:** Includes a variety of specialized explosives such as House TNT, MEGA TNT, Void TNT, and Meteor TNT[cite: 2].
* [cite_start]**Survival Ready:** All explosives come with balanced crafting recipes using vanilla materials like obsidian, ender pearls, and pistons[cite: 2].
* [cite_start]**Utility & Chaos:** Features both utility-focused explosives like **House TNT** (for quick structures) and high-power options like **4x TNT** or **Launch TNT**[cite: 2].
* [cite_start]**Custom Projectiles:** Introduces throwable explosives such as **Dynamite** and **Bombs** for more dynamic combat[cite: 2].

| TNT Type | Notable Ingredients | Purpose |

## 📋 Installation

1. Download the datapack folder.
2. Place it inside the `datapacks` folder of your Minecraft world.
3. Use `/reload` to enable the custom recipes and functions.
4. (Optional) [cite_start]Ensure the associated Resource Pack is active to see the custom 3D models and textures shown in the promotional art[cite: 1].

## 🔧 Technical Details

* **Version:** Compatible with modern Minecraft versions (verify specific `.json` tags).
* **Commands:** Access all items via the custom function: `/function jechu:give_all_tnt`.
